### 2. 프로젝트 소개
구매가 어려운 제품에대한 공동구매 요청을 할 수 있는 실시간 e커머스 플랫폼입니다.  
취미, 식료품, 패션, 뷰티 가리지 않고 해외 구매로 접근성이 떨어져 구입이 망설여지는 제품을 만나보세요!

### 3. 주요 기능

- **회원가입 / 로그인** - 소셜 로그인
- **마이페이지** - 상품 추가 요청, 리뷰관리, 주문 조회
- **관리자** - 상품 요청 관리, 공동 구매 관리, 주문 배송 관리
- **공동 구매 관련** - 상품 목록, 검색, 인기 상품, 참여자 현황, 리뷰 등
